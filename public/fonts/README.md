# Font Setup

This project uses Clash Display as the display font. 

To set up:
1. Download Clash Display Variable font from https://www.fontshare.com/fonts/clash-display
2. Place the .woff2 file in this directory as: ClashDisplay-Variable.woff2

Alternatively, you can use a different display font or use Google Fonts.
The fallback fonts (system-ui) will work if the custom font is not available.
